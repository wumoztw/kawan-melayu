使用說明

🚀 第一次使用（3 分鐘上手）
  1. 點右上角 ⚙️ 打開設定。
  2. 選擇 Provider（OpenRouter / Groq / Gemini / OpenAI）。
  3. 申請 API Key，貼到側欄的「API Key」欄位。
  4. 回到聊天區，在下方輸入框先輸入：你好。
  5. 接著照著 AI 的「你可以說」那一句念出來就好。
🔑 取得 API Key（必做）
本網站不內建金鑰，因為各家 AI 供應商都需要你自行註冊並建立金鑰。
OpenRouter Groq Gemini OpenAI
提示：第一次建議用 OpenRouter，模型先選 Auto 就能開始。
✅ 目標
用馬來文和 AI 對話，提升 Fluency，並維持 Confidence。
💾 存檔 / 讀檔
Save 會下載 JSON 存檔；Load 會還原進度。
提醒：如果要分享存檔，請取消勾選「存檔包含 API Key」。
📈 等級 / 場景 / 任務
Lv.1–3 以華語為主；Lv.4–6 轉以馬來文為主；Lv.7+ 幾乎全馬來文。
每次都會有任務推進，場景會依等級逐步解鎖。
🧩 常見問題
  • 看到「請先填入 API Key」：打開 ⚙️ 設定，把 Key 貼上後再送出。
  • 出現 401/403：通常是 Key 無效、貼錯、或該供應商沒有權限。
  • 出現 429：請求太頻繁或額度用完，稍等一下或換供應商。
  • 手機看不到側欄：按左上角 ☰，或按右上角 ⚙️。
